Stoopjs

The most simple model binder ever.

Welcome

Hi, I'm Stoopjs! My goal is to be the most simple model binder for html and javascript you will ever use. I have a tiny library, and a tiny file size. If you only need simple medel binding for your site, then check me out. I'm easy to learn and only take a few lines of code to get started.

Getting Started

Get started with Stoopjs in 3 easy steps.

  1. Download the script here
  2. Add it to your html file: <script src="../scripts/stoop.js"></script>
  3. Call stoop.apply({})
You are now applying an empty viewmodel to your page. Check out the documentation below to learn how to do something a bit more interesting.
