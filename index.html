Stoopjs

The most simple model binder ever.

Welcome

Hi, I'm Stoopjs! My goal is to be the most simple model binder for html and javascript you will ever use. I have a tiny library, and a tiny file size. If you only need simple medel binding for your site, then check me out. I'm easy to learn and only take a few lines of code to get started.

Getting Started

Get started with Stoopjs in 3 easy steps.

  1. Download the script here
  2. Add it to your html file: <script src="../scripts/stoop.js"></script>
  3. Call stoop.apply({});
You are now applying an empty viewmodel to your page. Check out the documentation below to learn how to do something a bit more interesting.

Documentation

First off, Stoopjs only works in browsers that support Object.observe. Currently that's only Google's Chrome browser. So, use at your own risk.

Below are all of the bindings that Stoopjs supports.

  • The `text` binding allows you set the text of a node.

    var viewModel = { someVal : 'Hi!' }; <span text="someVal"></span> Results in the following: <span text="someVal">Hi!</span>

  • An alternative to the `text` binding, you can put text inline like so:

    var viewModel = { someVal : 'Hi!' }; <span>|||someVal</span> Results in the following: <span>Hi!</span>

  • In order to show or hide elements use the `show` binding. If the expression evaluates to a falsy value, the element will be set to display: false, otherwise it will inherit its initial state.

    var viewModel = { someVal : false }; <span show="someVal"></span> Results in the following: <span show="someVal" style="display:none;">Hi!</span>

  • `val` will set the value property of an element.

    var viewModel = { someVal : 'hai' }; <input val="someVal" /> Results in the following: <span val="someVal" value="hai">Hi!</span>
